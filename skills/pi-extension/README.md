# pi-extension

Minimal skill for building and debugging Pi extensions and Pi skills.

## What It Covers

- Pi runtime paths vs local workspace paths
- Overlay sizing and focus defaults
- `pi-tui` rendering helpers for wrapping and width handling
- Hotkey and input handling patterns
- Safe sync of installed copies in `~/.pi/agent/`
- Fast verification for TypeScript-based extensions

## Use It For

- building a new Pi extension
- debugging a Pi overlay or hotkey issue
- fixing rendering, wrapping, or sizing bugs in `pi-tui` UIs
- creating or installing Pi skills
- checking whether the installed Pi copy matches the workspace copy

## Deliverable Style

The skill is intentionally short and operational. It is meant to give the agent a reliable workflow, not a full framework reference.
